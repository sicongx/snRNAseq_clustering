# snRNA-seq median+5SD QC, SCTransform / LogNormalize, clustree

Reproducible snRNA-seq pipelines for two projects — **scn1lab** and **dyrk1a**. Each Rmd reads 10x CellRanger matrices, runs QC and filtering (per-sample median + 5×SD nFeature max), doublet removal, SCTransform (with automatic LogNormalize fallback), Harmony integration, clustree, and clustering.

This repository is **code-only**. Place your own 10x filtered matrices under `data/<sample_folder>/` before running (see below).

## Repository layout

```
snRNAseq_median5sd_clustering/
├── scripts/                          # shared R helpers
├── scn1lab_snRNAseq/
│   ├── scn1lab_full_pipeline_median5sd_clustree.Rmd
│   ├── data/<sample_folder>/         # 10x matrices (local only)
│   └── out/                          # generated by Rmd
└── dyrk1a_snRNAseq/
    ├── dyrk1a_full_pipeline_median5sd_clustree.Rmd
    ├── data/<sample_folder>/
    └── out/
```

All paths are relative to the repository root (`snRNAseq_median5sd_clustering/`).

## Environment

- R ≥ 4.2, RStudio recommended
- ≥ 64 GB RAM suggested for Step 6 merged SCT (falls back to LogNormalize if SCT fails)

```r
install.packages(c(
  "Seurat", "dplyr", "ggplot2", "tibble", "patchwork",
  "harmony", "future", "clustree", "rmarkdown"
))
if (!requireNamespace("BiocManager", quietly = TRUE)) install.packages("BiocManager")
BiocManager::install(c("SingleCellExperiment", "scDblFinder", "glmGamPoi"),
                     update = FALSE, ask = FALSE)
```

## How to run

1. Place 10x matrices as described above.
2. In RStudio, set the working directory to `snRNAseq_median5sd_clustering/`.
3. Open one of the pipeline Rmds and **Run All** (or run chunks in order):
   - `scn1lab_snRNAseq/scn1lab_full_pipeline_median5sd_clustree.Rmd`
   - `dyrk1a_snRNAseq/dyrk1a_full_pipeline_median5sd_clustree.Rmd`
4. **Step 6** runs in three substeps (toggle flags in `pipeline-options`):
   - **6a** `run_step6a_sctransform <- TRUE` — SCT (or LogNormalize fallback) + Harmony + UMAP + neighbors
   - **6b** `run_step6b_clustree <- TRUE` — after 6a completes; review `clustree.png`
   - **6c** `run_step6c_clustering <- TRUE` — set `clustering_resolutions` from clustree, then cluster

Fixed QC thresholds: `nFeature_min = 200`, `nFeature_max = per-sample median + 5×SD`, `mt_max = 3`, scDblFinder singlets only.

Helpers live under `scripts/`: `repo_root.R`, `snRNAseq_qc_filter_helpers.R`, `snRNAseq_median5sd_clustree_helpers.R`, `snRNAseq_clustering_helpers.R`.
